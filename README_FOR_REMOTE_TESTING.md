## 🐛 常见问题

### 问题1：检测不到摄像头
**解决方案：**
- 确保USB摄像头已连接
- 关闭其他占用摄像头的程序（Skype、Teams等）
- 重新插拔USB摄像头

### 问题2：仍显示数字ID
**可能原因：**
- 系统中没有真实的USB摄像头设备
- 摄像头检测器无法获取设备名称

**这是正常的：** 如果没有真实的USB摄像头，系统会回退到显示默认名称

### 问题3：测试失败
**解决方案：**
- 检查Python环境和依赖包
- 查看详细错误信息
- 参考 `TESTING_GUIDE_FOR_REMOTE_COLLEAGUES.md` 获取详细指导
